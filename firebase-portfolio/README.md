# Arshia Music Portfolio - Firebase Edition

## 🎵 Features
- Matrix rain animation
- Neon cyberpunk design
- Firebase Firestore contact form
- Smooth scroll navigation
- Responsive design

## 🚀 Quick Start
1. **Firebase Setup:**
   ```
   Firebase Console → New Project
   Firestore Database → Create collection 'contacts'
   Project Settings → Copy config to script.js
   ```

2. **Test Local:**
   ```
   Live Server → index.html
   Console F12 → Submit form → Firestore new row
   ```

3. **Deploy:**
   ```
   GitHub repo → Settings → Pages → main/(root)
   Live: yourusername.github.io/repo
   ```

## 🔧 Firebase Config
Replace `firebaseConfig` in script.js with your project config.

## 📱 Live Test
Form → Submit → Firestore → instant row!
