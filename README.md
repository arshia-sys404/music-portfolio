# Arshia Music Portfolio\n\nMusic producer portfolio with beat upload, neon cyberpunk theme.\n\n![Neon Theme](https://via.placeholder.com/1200x600/000/FF1493?text=Arshia+Music)\n\n## Features\n- Beat upload/preview (Supabase Storage + DB)\n- Contact form (Supabase DB)\n- Responsive neon design\n- Smooth scroll navigation\n\n## Quick Start\n1. Open `index.html` in browser or use Live Server\n2. Supabase auto-loads beats\n\n## Supabase Setup\nProject: https://llocnfzgmqogfmrqobod.supabase.co\n\n**SQL for tables** (run in SQL Editor):\n```sql\n-- Beats table\ncreate table beats (\n  id uuid primary key default uuid_generate_v4(),\n  name text not null,\n  desc text,\n  url text,\n  created_at timestamptz default now()\n);\n\n-- Contacts table\ncreate table contacts (\n  id uuid primary key default uuid_generate_v4(),\n  name text not null,\n  email text not null,\n  message text not null,\n  created_at timestamptz default now()\n);\n\n-- Disable RLS for anon testing\nalter table beats disable row level security;\nalter table contacts disable row level security;\n```\n\n**Storage Bucket**:\n- Create bucket `beats` (public: true)\n\nKeys are in `script.js` (anon public - move to .env for production).\n\n## Deploy\n- Push to GitHub\n- Vercel/Netlify: Add env vars `VITE_SUPABASE_URL`, `VITE_SUPABASE_ANON_KEY`\n\n## Local Test\n```bash\nnpx serve .\n```\n\nRoll No: 25BCYC16 | Kristu Jayanti University
